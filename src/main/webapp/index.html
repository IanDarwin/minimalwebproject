Minimal: A Minimal Web App

This is a truly minimal web project for installation into a Java-powered application server. It may be of use:

  • In testing app deployers;
  • In testing app servers;
  • or as a basis for building a Java-based web site.

You can grab your own copy from my GitHub.

There is no documentation except the standard Java documents provided by Oracle and trillions of books, blogs, and web sites.
